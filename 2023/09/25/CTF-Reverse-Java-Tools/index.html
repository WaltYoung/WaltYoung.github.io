总结

此段代码的正向操作是先偏移值为3的凯撒加密，再进行移位

故反向操作应该是先进行移位，再偏移值为3的凯撒加密

具体步骤如下：

  1. 将String name = "$gourZroohK";的name变量的值，即$gourZroohK，进行移位，可得KhourZroog$
  2. 对KhourZroog$进行偏移值为3的凯撒解密，可得HelloWorld!
  3. 将HelloWorld!进行 MD5 计算后可得flag
